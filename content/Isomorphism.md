# [[Algebra]]
- 同态：
	- 设 V1 = ⟨A,◦⟩和 V2 = ⟨B,∗⟩是同类型的代数系统， f : A → B，并且∀x,y ∈ A,f(x◦y) = f(x)∗f(y), 称 f 是 V1 到 V2 的同态映射
	- **即用函数 f 从一个集合 A 映射到另一个集合 B，和群同态相同**
	- 单同态：f 是单射函数
	- 满同态：f 是满射函数
		- 称 V2 是 V1 的同态像，记为 V1 ∼ V2;
	- 自同态：f 是 V 到 V 的
	- 同构：f 是双射函数
