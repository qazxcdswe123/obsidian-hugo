- Special
	- `%rsp` : [[Memory|Stack]] Pointer (Location of runtime stack)
	- `%rip` : Instruction Pointer (Location of current code control point)
	- `%rax` : Store return value
- 32-bit
	- `%eax` through `%edx`
	- `%esi`, `%edi`, `%ebp`, `%esp`
	- `%r8d` through `%r15d`
- 64-bit
	- `%rax` through `%rdx`
	- `%rsi`, `%rdi`, `%rbp`, `%rsp`
	- `%r8` through `%r15`
