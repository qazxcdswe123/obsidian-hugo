# Ring

## Definition
加法：阿贝尔  
乘法：结合律  
整体满足分配律

## Properties
1. $a0 = 0a = 0$
2. $a(-b) = (-a)b = -ab$
3. $(-a)(-b) = ab$

重点：有分配率

# Integral Domain

## Definition
> 给定一个环 R，对任意元素 a ∈ R，如果存在元素 b ∈ R 使得 b ̸= 0 且 a ∗ b = 0，则称 a 为一个零因子 (zero divisor)。
> 如果交 换环 R 中没有除 0 以外的零因子，即 ∀a, b ∈ R，如果 ab = 0 则 有 a = 0 或 b = 0，则称 R 为整环。

# Subring

[[Subgroups]]
## Definition
> 给定环 R，R′ 是 R 的子集，如果 R′ 在环 R 的加法和乘法上也形成环，则称 R′ 是 R 的子环，记为 R′ ⊂ R。
