- [[CIDR]]  

| Property       | Behavior                                           |
| -------------- | -------------------------------------------------- |
| Datagram       | Individually routed packets. Hop-by-hop routing.   |
| Unreliable     | Packets might be dropped.                          |
| Best effort    | ..but only if necessary.                           |
| Connectionless | No per-flow state. Packets might be mis-sequenced. |
